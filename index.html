UMD Fall 2020 Course Information

This webpage was developed to help students get information about their courses for the upcoming Fall 2020 semester.

It accesses the information you can see by reading data directly from the University of Maryland's schedule of classes, which you can find here.

All of the information below will be up to date as of the time in the footer down below.

To search for more relevant information for your courses or department, go here.

All of the code that was used to do this can be found on this public GitHub page.

Please remember that this is a personal project and we aren't affiliated with UMD, so you should double check Maryland's schedule of classes before making any major decisions.
